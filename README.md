# Online Job Portal - Java Project

## Technologies Used
- Java
- JDBC
- MySQL
- HTML, CSS
- Apache Tomcat

## Modules
- User Registration & Login
- Job Posting
- Job Listing
- Apply for Jobs

## How to Run
1. Import project in Eclipse/IntelliJ
2. Configure MySQL database
3. Update DB credentials in DBConnection.java
4. Deploy on Apache Tomcat
